<h4>배포 이후 사이트 주소 집어 넣기</h4>
<h5>로딩중(LAWDING)은 판례 데이터를 기반으로 합리적인 법적 의사결정을 지원하는원스톱 법률 솔루션입니다.</h5>
<h5>의뢰인 상황에 최적화된 판례를 찾고 분석하여 적정 합의금, 승소 가능성, 소송 예상 기간을 제시합니다. 이를 통해 합리적인 의사결정을 돕고, 복잡한 법률 문서 초안 작성까지 원스톱으로 지원합니다.</h5>
